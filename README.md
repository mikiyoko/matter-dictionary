# ⚡ Matter 1.5 デバイス辞書

**by ライフテックコーディネーター 織田未来**

Matter 1.5仕様に基づく、スマートホームデバイスタイプ・クラスターの日本語リファレンスサイトです。

🌐 **公開URL**: https://mikiyoko.github.io/matter-dictionary/

---

## 📁 ファイル構成

```
matter-dictionary/
├── index.html          ← メイン辞書ページ（デバイスタイプ・クラスター一覧）
├── guide.html          ← 「Matterとは？」解説ページ ★ v1.1追加
├── README.md           ← このファイル
└── data/
    ├── clusters.js     ← ★ クラスターデータ（Matter仕様更新時に編集）
    ├── deviceTypes.js  ← ★ デバイスタイプデータ（Matter仕様更新時に編集）
    └── products.js     ← ★ 対応製品データ（随時追加）
```

## ✏️ データの更新方法

### Matter 1.6など新仕様が出たとき

1. `data/clusters.js` を開く
2. 新しいクラスターを配列に追加する
3. `data/deviceTypes.js` を開く
4. 新しいデバイスタイプを配列に追加する
5. GitHubにアップロードする → 自動でサイトに反映

### 対応製品を追加したいとき

1. `data/products.js` を開く
2. ファイル内のコメントに書いてある形式で製品データを追加する
3. GitHubにアップロードする → 自動でサイトに反映
